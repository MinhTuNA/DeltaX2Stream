1. Sơ lược về DeltaX2

Là một robot delta giá cả phải chăng nhưng mang lại hiệu quả cao. Ngoài mục đích sử dụng trong giáo dục và nghiên cứu, nó còn có thể được sử dụng trong các ứng dụng tự động hóa trong sản xuất. Delta X 2 là phiên bản nâng cấp của Delta X 1 về mọi mặt.

2. Hướng dẫn sử dụng trang điều khiển từ xa

Tải thư viện ở biểu tượng nơi góc phải màn hình

- Bấm vào Dowload Zip để tải xống thư viện

Sau khi tải xuống và giải nén ra và cho vào folder tùy ý, ta sẽ được 2 file như hình, bạn có thể mở file example để chạy thử hoặc tạo 1 file python cùng chung folder để code.

Mở file example.py bằng visual studio code

Bạn có thể chạy thử file example để xem kết quả (nhớ cài extension python nhé!):
